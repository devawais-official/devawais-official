## 👨‍💻 About Muhammad Awais

I am a **Senior Mobile Application Engineer** based in **Lahore, Pakistan**, with deep expertise in **native Android development** using **Kotlin and Jetpack Compose**, and cross-platform development with **Flutter and Dart**. I architect, build, and ship production-grade applications that are performant, testable, and maintainable at scale.

My engineering philosophy is grounded in **Clean Architecture**, **SOLID principles**, and **MVI/MVVM design patterns** — building systems that are easy to extend, easy to test, and built to last.

- 🏗️ **Architecture:** Clean Architecture · MVI · MVVM · Repository Pattern · Use Cases
- 📱 **Specialization:** Native Android (Kotlin, Jetpack Compose) · Flutter (Dart)
- 🌍 **Open Source:** Active AOSP contributor and open-source project maintainer
- ✍️ **Technical Writing:** Publishing Android & Flutter deep-dives on [Medium](https://medium.com/@devawais)
- 🌐 **Portfolio:** Case studies, projects & blog at [devawais.com](https://devawais.com)
- ⚡ **Currently:** Exploring advanced Compose optimizations and scalable mobile design patterns
